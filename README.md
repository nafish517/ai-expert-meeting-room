# AI Expert Meeting Room

Portfolio MVP: a voice-first meeting room where you talk to one AI expert at a time, while all experts share the same conversation context.

## Stack

- Next.js (App Router) + TypeScript + Tailwind
- Google Gemini (free tier)
- Browser Web Speech API (STT/TTS)
- Hosted on Vercel

## Local setup

1. Copy `.env.local.example` to `.env.local` and add your Gemini API key
2. Run `npm install`
3. Run `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)
